> Downdraft

A lightweight markdown viewer for macOS

About

Downdraft is a fast, native macOS markdown editor with live preview, syntax highlighting, Mermaid diagram support, and a distraction-free interface.

Features

  • Rendered preview with syntax-highlighted code blocks
  • Dark-themed editor with markdown syntax coloring and minimap
  • Mermaid diagram rendering
  • Folder sidebar with file tree navigation
  • Tabbed editing with drag-to-reorder
  • Command line tool for opening files from the terminal

System Requirements

macOS 26 or later.

Privacy

Downdraft does not collect, store, or transmit any personal data. All files are processed locally on your Mac. The app does not connect to the internet.
